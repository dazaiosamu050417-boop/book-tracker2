# Трекер прочитанных книг

Простое консольное приложение на Python для отслеживания прочитанных книг. Этот проект был выполнен в рамках учебного кейса для отработки навыков совместной разработки с использованием Git и GitHub.

## Возможности

- Добавление книги с указанием автора, названия, оценки (1-5) и даты прочтения.
- Защита от добавления дубликатов (автор + название).
- Просмотр списка всех книг.
- Расчет средней оценки по всем книгам.
- Статистика по количеству книг каждого автора.
- Удаление книги из списка.

## Как запустить

1.  Убедитесь, что на вашем компьютере установлен Python версии 3.6 или выше.
2.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/ВАШ_ЛОГИН/book-tracker.git
    cd book-tracker
    ```
3.  Запустите приложение:
    ```bash
    python main.py
    ```
